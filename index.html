Para começar

Entenda o que é o DOM. Isso é fundamental.

Além de Javascript, outros personagens serão HTML, CSS e SVG. Procure saber o que eles são

Para ver os exemplos, precisaremos de um servidor http. Sugerimos o live-server do node ou o SimpleHTTPServer em python. Tem instruções pros dois no README do repositório dessa introdução.

Os exemplos, em ordem

  • D3.js sem fazer nada demais ainda.
  • D3.js para criar elementos no dom com dados.
  • Formas básicas de SVG.
  • SVG + CSS.
  • Gerando SVG com D3.
  • Melhorando o gráfico de barras
  • Acessando dados fora do js
